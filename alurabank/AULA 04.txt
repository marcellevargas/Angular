- Como as duas Views criadas são parecidas
    eu vou criar um arquivo chamado View.ts para armazenar
    o código repetido.

- Quando criamos uma propriedade com o modificador private, ele não
    pode ser acessado pelas classes filhas, para resolver esse problema
    utilizamos o modificador protected que garante que as classes filhas
    acesso

- Eu posso tipar um tipo generico, basta colocar entre <T> ou qualquer outra letra.
  desse modo essa classe é sempre do tipo generico
