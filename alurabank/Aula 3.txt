-   Foi criado uma classe (Negociações) para encapsular
    toda a negociação feita

-   O objetivo da classe Negociações é armazenar um array
    de negociação. Para fazer isso, eu precisei declarar
    uma variável que irá receber um Array do tipo negociação

-   Para definir o tipo do Array utilizamos o símbolo <>
    então a minha variável recebe um Array<tipoDoArray>

-   Para escrever o tipo do array podemos escrever: 
    TIPO [] OU Array<TIPO> 

-   O "= []" significa que iremos inicializar um array vazio
